# rouletteboxd

don't know what to watch? press r to roll. picks a random film from any watchlist or list.

### how to install
1. install a userscript manager like [violentmonkey](https://violentmonkey.github.io/), [greasemonkey](https://www.greasespot.net/), or similar.
2. click [here](main.user.js?raw=true) to install.

### how to use
1. go to any letterboxd page with film posters (like your watchlist).
2. press **r** on your keyboard.
3. let fate decide.
4. watch the movie (or don't, i'm not a cop).

### credits
inspired by [gatleos/letterboxd-roulette](https://github.com/Gatleos/letterboxd-roulette) but rewritten from scratch for better ui, security, and performance.
